# NIKKE Crimson Night

A premium dark theme for Yazi inspired by **GODDESS OF VICTORY: NIKKE**.

## Features

- 🌑 Deep Dark UI
- 🔴 Crimson Accent
- 🟣 Midnight Purple
- 🔵 Navy Blue
- 📂 Custom Icons
- 💻 Comfortable for long coding sessions

## Installation

Clone the repository:

```bash
git clone https://github.com/SulthanAndyno/nikke-crimson.yazi.git
```

Copy the `flavor.toml` into your Yazi flavor directory and activate it in:

```toml
[flavor]
dark = "nikke-crimson"
```
